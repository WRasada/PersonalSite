The Journey Begins

What might that journey might be, you wonder? Well, if your here reading this then you probably guessed what it is but I can tell you it's going to change my life for the better. Programming, software engineering, coding, whatever you want to call it, that is what the journey entails. It's one I've dabbled in from time to time but never took seriously. While I regret not taking it seriously earlier on, all I can say is that its better late than never and its never too late.

So lets talk about what I've learned so far. For the past couple months i've learned some Ruby and Rails and built a small little web app, DevMatch. I spent a good amount of time the first month going through this course and nailing everything down. After that though, things slowed down a bit. Trying to find what to learn and just getting overwhelmed on just where to start. Last week, I can finally say I've found what I need to keep me going and I'm latching onto it. So within this week, I've learned quite a substantial amount. Here's a small list of what I learned:

  • HTML (Refresher)
  • CSS (Refresher)
  • Javascript
  • jQuery
and the list will keep growing. It also doesn't mean I'm an expert on those topics rather I know the basics. All I can do now is code everyday, which I've been doing this whole week and in the past 7 days have logged 42 hours while still working full-time. Some may say to not burn yourself out or you'll lose motivation. I can safely say that its not motivation keeping me going but discipline and I couldn't be happier. I've learned so much and I'm excited to keep learning and building up this website with everything that I learn. I started building this site this morning and I'm pretty happy with it so far. Still need more practice with design and layout. It's pretty hard just making something look clean and simple but that's where practice will help me. I'll be adding more javascript to the site to help with interactivity. For example this blog post can be minimized if you click the plus sign! Pretty cool! =D

I look forward to building my skills and eventually landing that developer job! Wish me luck and I'll update everyone next weekend! Probably some new changes to the site too!
